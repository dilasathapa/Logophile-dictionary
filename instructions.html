Instructions

  • Fill the grid with numbers from 1 to N, following the rules of Mathdoku.
  • The grid is divided into regions or boxes. It consists of rows and columns.
  • Each row and column must contain unique numbers from 1 to N.
  • use the play button to start your game
  • use the go back button to go to the main page
  • use the check solution button to check the solution
